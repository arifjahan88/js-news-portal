    What is the difference between var let and const in javascript?

    Var : Variables declared with var are in the function scope. Let : Variables declared as let are in the block scope. Const : Variables declared as const are in the block scope.

    what is the difference between regular function and arrow function in javascript?

    When a standard JavaScript function is called, the arguments keywords can be used to access the supplied parameters. However, arrow functions use the arguments from the outer function rather than having their own parameters.

    What is difference between foreach and map and filter in javascript?

    When using.forEach(), the same code is run on each entry of an array, however the array is left unchanged and the function returns undefined. A new array containing the changed items is returned by the function.map(), which runs the same code on each element of an array. Every element in an array is examined by.filter() to determine whether it satisfies a set of criteria, and it then returns a new array containing only the elements that do.

    why we use template string in javascript

    You can build strings in JavaScript using the ES6 feature known as Template String. Despite being primarily used for HTML or code embedding, Template Strings behave similarly to single and double quotes. Additionally, employing backticks makes string operations simpler.
